Welcome to TrICal's documentation!
==================================
Trapped Ion Calculator (TrICal) is a python package for simulating systems of trapped ions.



Installation
============
Clone the `TrICal github repository <https://github.com/QITI/TrICal.git>`_ using the following command::

    git clone https://github.com/QITI/TrICal.git



Tutorial
========
Run the following commands in the root of `TrICal github repository <https://github.com/QITI/TrICal.git>`_ for the tutorial::

    cd tutorial
    jupyter notebook tutorial.ipynb
